## Mac安装

不出意外下载后打开会出现这个情况”**应用打不开或提示损坏怎么办**“， 解决办法如下

1. 打开终端（在实用工具中找到），并输入以下命令以开启外部来源应用的权限。

   `sudo spctl --master-disable`
